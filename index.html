Kim jestem


Jestem absolwentem studiów ekonomicznych, który postanowił poszerzyć swoje kompetencje zawodowe, zgłębiając zupełnie nową dziedzinę. Naukę programowania rozpocząłem na własną rękę, pod koniec 2018 roku. Jestem obecnie skupiony na poznawaniu technologii związanych z pisaniem aplikacji webowych i front-endem. Pracując lubię podejmować zróżnicowane zadania, z łatwością przyswajam nowe umiejętności i wiedzę, posługuję się swobodnie językiem angielskim oraz dobrze odnajduję się w pracy z innymi ludźmi.

netFilmoteka

Kod strony
Portal filmowy umożliwiający przeglądanie, zapoznawanie się z fabułą czy porównywanie ocen filmów, znajdujących się w bazie filmowej TMDb.

Charakterystyka:

  • wyszukiwanie filmów po tytule (w różnych językach)
  • wyszukiwanie filmów z danego gatunku
  • sortowanie filmów po wybranym kryterium
  • możliwość sprawdzenia szczegółów filmu

Technologie:

  • HTML i CSS (flex-box, grid, media queries)
  • JavaScript
  • React (React-Router-DOM)
  • Redux (Thunk)
  • Axios
  • TMDb API
